Review of all HTML tags that modify text

                
    Text Tags
    
                
    Underlinded
    
                    
  • abbr (for abbreviation, dotted under line)*
    • 
                    
  • dt (defines a term/name in a description list)*
    • 
                    
  • ins (underlines the word)
    • 
                    
  • u (for underline)
    • 

                
    Bold/Font-Weight and Size
      
                    
  • b (for bold, an alternative would be font-weight in CSS)
    • 
                    
  • h1-h6 (effects the font-weight and size)*
    • 
                    
  • strong (used to define text with strong importance)
    • 

                
    Repositioning & Font-Size
    
                    
  • bdo (Bi-Directional Overrid, aka change the direction of the letters)
    • 
                    
  • blockquote (doesn't change the font, it repositions it with indentation)*
    • 
                            in Bootstrap 3, you can show the quote on the right 
                        just by calling the class .blockquote-reverse
                            
                    
  • small (font-size change, used for copyrights and other side-comments)*
    • 
                    
  • sub (for subscript, appears half a character below the normal line, with reduced font- used for chemical formulas)
    • 
                    
  • sup (for superscripted text, appears a char above the normal line, with reduced font- used for footnotes)
    • 

                
    Line through word(s)
    
                    
  • del (puts a line through the word)
    • 
                    
  • s (for strike? puts a line through the word(s) just like del)
    • 
                    
                
    Surrounds word(s)
     
                    
  • mark (this highlights the word(s), in a default yellow)*
    • 
                    
  • q (for quote)
    • 

                
    Italic
    
                    
  • address (italic)
    • 
                    
  • cite (makes the text italic)
    • 
                    
  • dfn (stands for definition, and also italisizes the word being defined)
    • 
                    
  • em (stands for emphisized, and also italisize the word)
    • 
                    
  • i
    • 
                    
  • var (defines a variables in programming or in a mathematical expression, italic)
    • 

                
    Monospace Font
    
                    
  • code (monospace)*
    • 
                    
  • pre (monospace)*
    • 
                    
  • samp (is used to define sample output from a computer program, turns the word(s) to monospace font
    • 
                    
  • kbd (stands for key bord, and it puts the word(s) in computer type font- monospace font with a background-color)*
    • 
                


                *indicates that they are used in Bootstrap 3
